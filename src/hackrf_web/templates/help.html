1. 无线电基础

频率(Frequency)是电磁波每秒振荡的次数,单位赫兹(Hz)。收音机"调台"就是选频率。频率与波长成反比:波长 λ(米)≈ 300 ÷ 频率(MHz)。例如 433.92 MHz 波长约 0.69 m,2.4 GHz 约 12.5 cm——这决定了天线尺寸。

常见频段(ISM / 民用)
频段典型用途
315 MHz北美车钥匙、胎压、车库门
433.92 MHz欧亚车钥匙、遥控插座、气象站、门磁(最热闹的 ISM 段)
868 / 915 MHz欧洲/美洲 ISM、LoRa、智能电表
1575.42 MHzGPS L1 民用信号
2.4 GHzWiFi、蓝牙 / BLE、ZigBee
调制(Modulation)—— 信息怎么"骑"在电波上
  • OOK / ASK(通断 / 幅移键控):有载波=1、没载波=0。绝大多数便宜遥控器用它,波形像莫尔斯电码。
  • FSK / GFSK(频移键控):用两个略不同的频率代表 0/1。抗噪比 OOK 好,蓝牙、许多车钥匙用它。
  • PSK / QAM:用相位/幅度组合承载更多比特,WiFi、数字电视等高速链路用。
2. SDR 与 IQ 采样

软件定义无线电(SDR)把传统电台里焊死的解调电路搬进软件:硬件只负责把天线信号搬到基带并数字化成一串数字,之后的滤波、解调、解码全由软件完成。所以同一台 HackRF 今天当遥控接收机、明天当频谱仪、后天当 GPS 发射机。

IQ 数据

SDR 用两路正交分量 I(同相)与 Q(正交)来完整描述一个信号的幅度与相位。可理解为把信号画在复平面上的坐标 (I, Q)。HackRF 每个采样输出一对 8-bit 有符号的 I、Q,交错存储:I₀ Q₀ I₁ Q₁ …,即每采样 2 字节。这就是录制的 .raw 文件内容。

采样率与带宽

采样率(Sample rate)决定你一次能观测多宽的频谱。由奈奎斯特定理,采样率 Fs 能覆盖约 Fs 宽的带宽(复采样)。8 Msps ≈ 看 8 MHz 宽。基带带宽(-b)是硬件模拟滤波器宽度,一般设为略小于采样率以滤除带外噪声。

文件大小 = 采样率 × 2 字节 × 秒数。例:8 Msps 录 10 秒 = 8M×2×10 = 160 MB。窄带信号(如钥匙)用 2 Msps 足矣,文件小 4 倍。
3. 功率、dBFS 与增益

dBFS(dB relative to Full Scale)是数字域的功率刻度:0 dBFS 是 ADC 满量程(最强),数值越负越弱。它是相对值,不是绝对功率(dBm)。

  • 底噪(Noise floor):没有目标信号时的背景能量水平,是判断"有没有信号"的基线。HackRF 典型底噪在 −25 ~ −40 dBFS 量级(随增益变化)。
  • 动态范围:最强信号与底噪之差。HackRF 是 8-bit ADC,理论动态范围约 48 dB,有限——所以要靠增益把目标信号推到合适电平。
三级增益(接收链路)
级参数范围作用
RF Amp-a0/1(~+11 dB)最前端总开关,信号极弱时开;强信号别开,易过载
LNA(IF)-l0–40,步进 8低噪声放大,靠近天线,主要影响灵敏度
VGA(基带)-g0–62,步进 2后级基带增益,把信号抬到 ADC 合适电平

发射侧只有一级 TX VGA(-x,0–47)加可选 RF Amp。调增益的目标:目标信号明显高出底噪,但峰值不要顶到 0 dBFS(过载会失真、产生假信号)。

4. 看懂频谱与峰值

实时频谱:横轴频率、纵轴强度(dBFS),显示各频点此刻有多强。扫频是从低到高逐段扫过去的,那条青色竖线是当前扫描位置。

峰值保持(Peak Hold):叠加一条暗黄包络,记录每个频点出现过的最强值。间歇/突发信号(按一下钥匙)也会被定格,方便数清楚一共有几个信号、分别在哪个频点——这是找信号最实用的手段。

识别信号:遥控器按下时,会在其载波频点出现一个明显高出底噪的窄峰;OOK 信号在时域上是断续的,所以峰值保持能抓到它;WiFi/蓝牙则是宽而杂的一片。

配合本工具:选好频段 → 连续扫描 + 峰值保持 → 按下目标遥控 → 在峰值列表读出频点 → 点『抓包』直接带着这个频点去录制。

6. 抓包与重放原理

重放攻击(Replay Attack):把目标发射的原始射频(IQ)完整录下来,之后原样再发射一遍,让接收方以为是"真"的指令。它不需要理解协议内容——是"照抄"而非"破解"。

为什么固定码能成、滚动码常常不行
  • 固定码:每次按键发的码字都一样。录一次就能无限重放。老式车库门、便宜遥控插座、部分老车属此类。
  • 滚动码(Rolling code,如 KeeLoq):每次按键码字按算法递增,接收方维护一个计数窗口。重放旧码通常已"过期"被拒。需要更复杂的手法(且常涉及法律红线),本工具不做。
成败命门:采样率必须一致。重放用的 -s 必须与录制时相同,否则信号在时间轴上被拉伸/压缩,接收方无法解码。本工具在重放时已把采样率锁定为录制值。

录制时看实时功率:安静时是底噪,目标一发射,平均功率明显跳升——这是"录到有效信号"而非空录的判据。

7. GPS 欺骗原理

GPS 接收机通过测量多颗卫星信号的到达时间差解算自身位置。若用 SDR 合成一组更强的、内部自洽的伪造卫星信号,接收机会锁到伪造信号,报出攻击者设定的位置——这就是 GPS 欺骗(Spoofing)。

  • L1 C/A:民用 GPS 信号,中心频率 1575.42 MHz,无加密,是欺骗的对象。
  • 星历(Ephemeris):卫星轨道参数(RINEX 文件)。gps-sdr-sim 用它+目标坐标+时间,算出该时该地各卫星应有的信号并合成 IQ。真实接收机要锁定,星历需与当前 GPS 时间匹配。
  • 时钟精度:GPS 对频偏敏感,克隆板晶振漂移可能导致锁不上,需实测调整。

本工具的做法:在地图上点多个坐标、各设停留时长 → 逐点用 gps-sdr-sim 合成信号 → 循环发射,让接收机的位置按时间跳变。地图支持高德(国内快,坐标自动 GCJ-02→WGS-84 纠偏)与 OpenStreetMap 两种底图。

获取当天广播星历(手动下载) —— GPS 页可先试"自动获取";失败时到下列任一处下载 brdcDDD0.YYn.gz(DDD=年积日,YY=年份后两位),解压后在 GPS 页上传:
  • NASA CDDIS(需免费 Earthdata 账号):https://cddis.nasa.gov/archive/gnss/data/daily/
  • IGS 德国 BKG 镜像:https://igs.bkg.bund.de/root_ftp/IGS/BRDC/
  • 法国 IGN:https://igs.ign.fr/pub/igs/data/
必须在射频屏蔽箱内进行。向开放空间发射 GPS 会干扰周边真实导航与授时基础设施,违法且危险。

9. 安全与合规
  • 本工具定位侦察 + 授权重放。接收/扫描一般合法(被动);发射(重放、GPS)只可用于你拥有或获得书面授权的设备。
  • GPS 等高危发射必须在射频屏蔽箱内,不得向开放空间辐射。
  • 无线电发射受当地法规约束,请在合规前提下测试。滚动码去同步、干扰阻塞等破坏性手法不在本工具范围内。
10. 快速上手:重放一把授权钥匙
  1. 确认频率 —— 频谱扫描页选"汽车遥控 433MHz",连续扫描+峰值保持,按住钥匙,在峰值列表读出频点(如 433.92 MHz),点该行『抓包』。
  2. 录制 —— 抓包页已带入频点;采样率选"窄带 2 Msps",时长 3 秒,点开始的同时按住钥匙,看实时功率跳升。
  3. 重放 —— 抓包库里点重放,参数已自动回填(采样率锁定),天线靠近接收端,确认发射。固定码通常即可复现。
